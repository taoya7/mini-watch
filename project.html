简介

Claude Code 在后台跑长任务时，你很难知道它是在干活、卡住要授权、还是已经做完。本项目用一盏看得见的红绿灯把这件事实体化：Agent 通过 hook 上报灯语，服务端聚合后经 WebSocket 实时推到手机和 Web 控制台；同一条通道还顺带把 Mac 的 CPU / GPU / 功耗 / 风扇喂给手机展示。

移动端

./iapp

Flutter（iOS）。红绿灯页用 GLSL 三盏灯呈现状态，另有 Mac 系统信息页（风扇 / 地球等 shader 动效）。

  • 绿灯常亮 = 空闲 / 正在工作
  • 黄灯慢闪 = 需要关注
  • 红灯快闪 = 阻塞 / 等授权
  • WebSocket 直连服务端，状态即时刷新

后端 + 控制台

./iapp/server

Hono API + Vue 3 SPA，pkg 打包成单个可执行文件，pm2 一键启动（端口 4600）。

  • /api/signal-light/state 收 Claude hook 上报
  • 多会话灯语聚合，最危险的状态优先
  • WebSocket 广播到手机与控制台
  • 控制台在 src/client/，shadcn-vue + Tailwind v4

数据采集

./macmon

基于 vladkens/macmon 的最新版本修改，新增风扇转速统计。Rust 编写，借私有 macOS API 免 root 读取 Apple Silicon 的实时性能数据。

CPU / GPU 占用功耗温度内存风扇 RPMsudoless

移动端部署iOS 真机安装

配置 API 地址
打包前先改 iapp/lib/config.dart 的 apiBaseUrl 为你的后端地址，WebSocket 地址会据此自动推导（https→wss）。
make 打包 ipa
根目录执行，产出未签名安装包 dist/app.ipa。 # 默认目标即 unsigned
make
签名并安装到设备
用你的开发证书 / 描述文件对 dist/app.ipa 重签名，再经 Xcode、Apple Configurator 或 sideload 工具装到 iOS 真机即可。
